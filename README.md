## 📖 About

This repository contains **18+ ready-to-use prompt templates** covering all major QA domains. Whether you're writing test cases, automating tests, analyzing bugs, or planning test strategies, you'll find practical prompts to accelerate your work with AI tools like ChatGPT, Claude, Gemini, and more.

## 💡 Example Usage

### Generate Test Cases
```
Generate comprehensive functional test cases for a User Registration feature with the following requirements:
- Username must be 6-20 characters
- Email must be valid format
- Password must be 8+ characters with at least 1 uppercase, 1 lowercase, 1 number, 1 special character
- Age must be 18+

Include:
1. Positive test scenarios
2. Negative test scenarios
3. Boundary value analysis
4. Equivalence partitioning

Format as: Test Case ID | Description | Preconditions | Test Steps | Expected Result | Priority
```

### Create Automation Script
```
Generate a Selenium WebDriver test script in Java for login functionality using Page Object Model pattern, 
including explicit waits, assertions, and error handling. Framework: TestNG
```
